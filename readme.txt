== Description ==

This plugin can create an inner loop inside the main loop of a wordpress page. This inner loop allows user to iterate over the child page objects of the main page. 


== Installation ==

1. Upload `the-subpage-loop.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the following code in the page.php in your themes. Make sure that the code snippet is inside the main WP    LOOP.  while(have_subpages()): 
            the_subpage_title();
            the_subpage_content(); 
          endwhile;
